# Atmo Weather (iOS Application)

## Project Description

Atmo Weather is a complete weather app solution for the everyday user. With live weather data supplied by OpenWeather, Atmo Weather 
streamlines information in a meaningful way. Atmo provides weather across multiple custom locations selected by the user. View weather data
for the current week, day and even hour, in addition track data such as Wind speed and dirction, UV Index and humidity. Atmo's latest update 
introduces Atmo Cycles, a quick and easy summary of moon cycles for the coming week.


## App User Features
- Atmo uses the users current location to provide accurate current weather data
- Users may add their own locations to Atmo through search
- Autocomplete search functionality for custom locations
- Weather information by week, day and hour
- Additional Weather information: Wind direction, wind speed, UV Index, and humidity
- Dynamic backgrounds that match the locations current weather
- Animated weather effects to suit the current conditions
- Atmo Cycles - Introduces weekly summarized moon data

## Technologies Used
- RESTful Api handling (with use of OpenWeather: https://openweathermap.org/api)
- App based on the MVVM-C Architecture
- Apple Mapkit
- Core Location & Reverse Geolocation
- Swift
